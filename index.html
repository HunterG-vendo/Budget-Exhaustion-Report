Quick Start Guide
1
Pull the Budget Report — Campaign Manager → Reports → Sponsored Products → Report type: Budget → Time unit: Daily → Last 7 days → Download as .csv
2
Pull the Campaign Report — Same place → Report type: Campaign → Time unit: Hourly → Last 7 days → Download as .csv
3
Upload both files above and click Generate Report
4
Select your timezone in the top right — times in the report will adjust from Amazon's Pacific time to yours
5
Click any campaign on the left to see its hourly spend curve and heatmap — red bars = budget exhausted, amber = nearly there
Campaigns flagged = spend ≥ 95% of daily budget
